# Example Calculations

**Calculate $\mu$**

$$\mu = \frac{m}{L}$$
$$\mu = \frac{2.29*10^{-3}}{1.203}$$
$$\mu = 1.9*10^{-3}$$

## Calculate Wavelength ($\lambda$)

**$n = 7$**

$$\lambda = \frac{2L}{n}$$
$$= \frac{2(1.203m)}{7}$$
$$\lambda = 0.344 \space m$$

**$n = 6$**

$$\lambda = \frac{2L}{n}$$
$$= \frac{2(1.203m)}{6}$$
$$\lambda = 0.401 \space m$$

**$n = 5$**

$$\lambda = \frac{2L}{n}$$
$$= \frac{2(1.203m)}{5}$$
$$\lambda = 0.481 \space m$$
**$n = 4$**

$$\lambda = \frac{2L}{n}$$
$$= \frac{2(1.203m)}{4}$$
$$\lambda = 0.602 \space m$$


**$n = 3$**

$$\lambda = \frac{2L}{n}$$
$$= \frac{2(1.203m)}{3}$$
$$\lambda = 0.802 \space m$$


## Calculate Tension ($T$)

**$n=7$**
$$T = mg$$
$$= (0.214)(9.8)$$
$$T=2.1 \space N$$

**$n=5$**
$$T = mg$$
$$= (0.688)(9.8)$$
$$T=6.7 \space N$$

**$n=3$**
$$T = mg$$
$$= (1.190)(9.8)$$
$$T=11.7 \space N$$

## Calculate Frequency ($f$)

**$n=7$**

$$f = \frac{1}{\lambda}\sqrt{\frac{T}{\mu}}$$
$$f = \frac{1}{0.344}\sqrt{\frac{(2.1)}{(1.9*10^{-3})}}$$
$$f=96.5 \mbox{ Hz}$$

**$n=5$**

$$f = \frac{1}{\lambda}\sqrt{\frac{T}{\mu}}$$
$$f = \frac{1}{(0.481)}\sqrt{\frac{(6.7)}{(1.9*10^{-3})}}$$
$$f= 123.7\mbox{ Hz}$$

**$n=3$**

$$f = \frac{1}{\lambda}\sqrt{\frac{T}{\mu}}$$
$$f = \frac{1}{(0.802)}\sqrt{\frac{(11.7)}{(1.9*10^{-3})}}$$
$$f= 97.8\mbox{ Hz}$$
