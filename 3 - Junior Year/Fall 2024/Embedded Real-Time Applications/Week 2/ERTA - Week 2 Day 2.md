# Digital Logic Review

## One’s Complement

```ad-summary
title: Remember
1. Convert the number to binary
2. Then if it's positive, leave it be.
3. If it's negative:
	1. Invert every bit in the number
```


## Two’s Complement

```ad-summary
title: Remember
1. Convert the number to binary
2. Then if it's positive, leave it be.
3. If it's negative:
	1. Invert everybit in the number
	2. Add one
```
