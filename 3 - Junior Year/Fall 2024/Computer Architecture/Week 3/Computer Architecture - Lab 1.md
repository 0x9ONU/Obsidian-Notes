# The Memory System

## Pointer System

- `0x0` = `zero` = 0
- `s` = saved registers
- `t` = temporary registers
- `a` = arguments
- `gp` = global pointer
- `sp` = stack pointer
- `ra` = return address
- `tp` = text or thread pointer
